# object-detection-with-opencv
object detection with opencv

# Required packages
```
opencv-python
imutils
```

# Run
execute the following command from your venv
```
python filter_object_detection.py --prototxt MobileNetSSD_deploy.prototxt.txt --model MobileNetSSD_deploy.caffemodel
```
